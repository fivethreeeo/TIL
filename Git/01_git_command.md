## Git
- VCS 중의 하나 (Version Control System)
- CVC의 단점을 보완하기 위해 (Centralized Version Control)
  - 서버에 업데이트해서 관리
  - 서버에 문제가 생기면 일을 못함
  - 오프라인에서 일을 못함
- DVC(Distributed Version Control)
  - 히스토리가 서버에만 있지 않음
  - 개발자들이 동일한 히스토리를 보유
  - 서버에 문제가 생겨도 보완 가능
- 프로젝트의 전체내용을 스냅샷해서 보유
  - 버전 사이 , 브랜치 사이 이동이 쉽고 빠름
  - 버전 관리 용이
  - 브랜치 관리로 효율적인 협업 가능
